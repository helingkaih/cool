web Worker 使用说明

什么是 web worker

在讲解之前，我们还需要了解一个概念，线程;

JS 是单线程的工作环境，如果一个 JS 任务中，需要完成极大的数据处理，那么整个 JS 线程将会出现阻塞现象，进而会导致之后的 UI 和 IO 停止运行，表现有：浏览器卡死、鼠标键盘操作没反应等，这就是很明显的线程阻塞现象

JS 是单线程的但是浏览器是多线程的，也就是说我们可以开辟子线程完成 大量的 JS 处理，这就要用到 web Worker 了

web Worker 与主线程之间只有简单的通讯，不会共享任何资源
